## Step 3 : Creating a key-value pair RDD 
Use `map` to create pair RDD

#### Scala:
```scala
val userMeetupsKV = meetups.map(line => {
                    val tokens = line.split(",") // split the line
                    (tokens(0), tokens(1)) // create a KV pair (user, meetup)
                    })
```
