== STEP 1)  login to the Spark node
instructor will provide details


== STEP 2) Getting familiar with the VM

Installed software
  JDK 7
  Scala 2.11
  SBT
  Spark 1.1
  Maven

File Layout
  home dir (/home/ec2-user)
    +-- spark  :  spark distribution
    +-- spark-labs :  spark labs
    +-- data   : pointer to spark-labs/data


== STEP 3) update spark-labs
  $   cd ~/spark-labs
  $   git pull


== STEP 4) Generate data files
  $   cd ~spark-labs/data/twinkle
  $   ./create-data-files.sh

This will create text files of various sizes
  1M ,   10M,   100M,   500M  and 1 G
We will use these files for testing
